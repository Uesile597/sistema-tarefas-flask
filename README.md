# Sistema de Tarefas em Flask

Aplicação web desenvolvida com Flask para gerenciamento de tarefas, incluindo cadastro de usuários, autenticação, controle de tarefas e dashboard com gráficos.

## Funcionalidades
- Cadastro e login de usuários
- CRUD de tarefas
- Status de tarefas
- Dashboard com gráficos
- Integração com banco de dados MySQL

## Tecnologias utilizadas
- Python
- Flask
- SQLAlchemy
- MySQL
- HTML, CSS e JavaScript

## Como executar o projeto
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
